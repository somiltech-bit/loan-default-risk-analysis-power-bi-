# Dashboard Guide

## Overview

This Power BI dashboard was developed to analyze loan default behavior, borrower demographics, and financial risk indicators using 255,348 loan records.

The dashboard is divided into three sections to help financial institutions identify risk patterns, monitor lending trends, and support data-driven decision-making.

---

# Dashboard Pages

## 1. Loan Default & Overview

### Objective

Provide a high-level summary of loan distribution and default trends.

### Visuals Included

* Loan Amount by Purpose
* Average Income by Employment Type
* Default Rate (%) by Employment Type
* Average Loan Amount by Age Group
* Default Rate (%) by Year

### Key Analysis

This page helps answer:

* Which loan purposes receive the highest funding?
* Which employment groups have higher default rates?
* How does age influence loan amounts?
* How have default rates changed over time?

---

## 2. Applicant Demographics & Financial Profile

### Objective

Analyze borrower demographics and financial characteristics.

### Visuals Included

* Median Loan Amount by Credit Score Category
* Average Loan Amount by Age Group and Marital Status
* Total Loan Amount by Credit Score Bins
* Loan Distribution by Mortgage and Dependents
* Number of Loans by Education Type

### Key Analysis

This page helps answer:

* How does credit score impact borrowing behavior?
* How do age groups differ financially?
* How do marital status and education affect loan patterns?
* Which borrower segments contribute most to total loan value?

---

## 3. Financial Risk Metrics

### Objective

Monitor financial performance and identify risk trends over time.

### Visuals Included

* Year-over-Year (YOY) Loan Amount Change
* Year-over-Year (YOY) Default Loan Change
* YTD Loan Amount by Credit Score Bins and Marital Status
* Total Loan Amount by Income Bracket
* Total Loan Amount by Employment Type

### Key Analysis

This page helps answer:

* How have loan amounts changed over time?
* How have default patterns evolved?
* Which income groups receive the largest loans?
* Which employment categories contribute most to loan value?

---

# KPIs and Metrics Used

The dashboard uses custom DAX calculations, including:

* Default Rate (%)
* Average Loan Amount
* Median Loan Amount
* Average Income
* Total Loan Amount
* Year-to-Date (YTD) Loan Amount
* Year-over-Year (YOY) Loan Amount Change
* Year-over-Year (YOY) Default Loan Change

# How to Use

1. Open `loan_default_dashboard.pbix` in Power BI Desktop.
2. Navigate between the three dashboard pages.
3. Explore borrower segments and risk patterns.
4. Use visuals to compare trends across years, demographics, and financial profiles.
